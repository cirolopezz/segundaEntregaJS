Una funcion constructura permite ingresar por prompt propiedades del objeto alumno. Entre estas propiedades se ingresan notas para calcular el promedio que cada alumno posee e incorporarlo en el objeto. Estos objetos se incorporan al array denominado matricula, a fin de poder operar con los metodos que estan asociados a array. En este caso solo se instancian 3 objetos en el array y se utiliza la funcion filter para encontrar alumnos que coincidan con el string de entrada por prompt y para encontrar los alumnos aprobados.
